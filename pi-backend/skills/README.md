# Multi-Agent Skills

Place custom skill definitions here. Each skill should be a directory containing
a SKILL.md file following the pi skill format.

Skills in this directory are loaded by the Multi-Agent system at runtime
(via the `SKILL_ROOT` path configured in monitor-server.js).

To add a skill:
1. Create a subdirectory: `mkdir my-skill`
2. Add `my-skill/SKILL.md` with the skill definition
